# 🛡️ Cybersecurity Labs: Bash & Linux Offensive Skills

Bienvenido a mi repositorio personal de laboratorios de ciberseguridad. Este espacio está dedicado a proporcionar entornos controlados y realistas diseñados para poner a prueba y mejorar habilidades técnicas centradas en **Linux**, scripting avanzado en **Bash** y tácticas ofensivas/defensivas relacionadas.

## 🎯 Objetivo del Repositorio
El propósito de estos laboratorios es ofrecer escenarios "hechos a mano" que simulan situaciones reales de ciberseguridad. Aquí no encontrarás solo vulnerabilidades típicas, sino retos orientados a:
* **Living off the Land (LotL):** Explotación y post-explotación usando herramientas nativas del sistema.
* **Hardening & Misconfigurations:** Identificación de errores de configuración en entornos Linux.
* **Automatización:** Uso de Bash para tareas de enumeración, movimiento lateral y persistencia.
